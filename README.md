# Webpage-Lofthouse

Добро пожаловать в репозиторий сайта лендинга о жилом комплексе Lofthouse!

## О проекте

Landing page.

## Как установить

```

npm i
gulp

```

Если возникнет ошибка Error: Cannot find module 'webp-converter/cwebp'

```

npm i webp-converter@2.2.3 --save-exact

```
